Hi! Thank you so much for downloading Life Simulator! 

To run the game, simply open home.html on your browser. This will bring you to the home page where you can begin your adventures in another life!

Life simulator recommends that you use the latest version of Google Chrome to run this game. Firefox cannot be used as Local Storage data, which is the system used to store data within the game, does not transfer between webpages. Other browsers may work, but have not been tested.
